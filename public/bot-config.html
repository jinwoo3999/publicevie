Trạng Thái Bot Chưa Kích Hoạt

💡 Lưu ý: Bot cho phép check tài khoản realtime từ hậu đài của bạn. Nếu không dùng bot, hệ thống sẽ check trong danh sách đã sync thủ công.
URL endpoint của bot để check tài khoản
API key để xác thực với bot (nếu có)

📖 Hướng Dẫn

Bước 1: Chuẩn bị bot của bạn với API endpoint

Bước 2: Bật "Bật Bot" và nhập thông tin

Bước 3: Test kết nối để đảm bảo bot hoạt động

Bước 4: Lưu cấu hình

Bước 5: Khi check tài khoản, hệ thống sẽ tự động gọi bot

📝 Format API Bot:
Request: POST /check
Body: { "player": "username" }
Response: { "exists": true/false }
